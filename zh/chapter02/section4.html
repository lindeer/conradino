行政班子

行政班子

就目前而言，乌尔里希的两个赞助人之间的矛盾还没有公开化。在莱茵短暂停留后返回家乡的理查当然不想在德国留下任何公开的敌人，实际上他还想通过自己的利益来收买路德维希，在离开德国直到他回来前，他还短暂地将迪林根伯爵(Grafen von Dillingen)移交给王国的封地转给了路德维希。但是随着斯陶芬派活动的猖獗，冲突紧接着就爆发了。

在此之前，路德维希试图通过保护士瓦本公国来给他外甥一个更加稳定的地位。那里的教会和领主厌倦了等待他们的国王阿方索，不过也不像其它挺德派公国那样转而支持理查而是保持中立。路德维希与他们进行了谈判，尤其是与康斯坦茨主教埃伯哈德(Eberhard von Konstanz)，结果令人满意。当时十岁的康拉丁本应正式接管士瓦本的统治，但目前在康斯坦茨主教的辅佐下临时主持那里的事务，如果是路德维希自己接管，就会引起贵族对他的怀疑和反抗。

1262年春天，康拉丁去了他的实际公国，并于8月初先后在乌尔姆(Ulm)和罗特韦尔(Rotweil)举行了两次庭审。在确认了祖先授予的罗滕明斯特修道院(Kloster Rottenmünster)附近的权利后，他继续前进并在康斯坦茨(Konstanz)和他的舅舅会合。在这里，我们第一次得以了解那些围绕着年轻公爵的名字，在这样一个未知的非常的环境下，可以好好审视一下他们。

士瓦本的伯爵们没有像今天这么多聚集在康拉丁周围了。他们一个个都表示了对公爵的赞赏。除了图宾根的帕拉蒂尼伯爵(Pfalzgrafen von Tübingen)、索伦的弗里德里希伯爵(Friedrich von Zollern)之外，我们还发现乌尔里希·符腾堡(Ulrich von Würtemberg)的身影，他们给予帮助和建议，同时也在寻求新的恩惠，这次是金钱方面的。符腾堡的出现恰恰表明，在这类人中间康拉丁为他的公爵权力是找不到任何长久的支持的。这些伯爵面对普遍的混乱和法律的含混，正在逐渐让自己成为主人，他们可没有一点兴趣头上再来个统治者。新公爵只是个孩子，使他们更容易达成一致；要是这个男孩能戴上德国王冠，那与王权的联系可以反过来保护他们免受其他统治者侵犯。他们愿意提供支持，但并不是在没有相应好处的情况下。据我们所知，后来康拉丁只是与他们中的很少几个人有着紧密的关系。

小贵族们经常在竞争中失利，所以更关心公爵权力的巩固，他们需要从中获得支持。赫尔曼·亨海姆(Hermann von Hürnheim)在里斯(Ries)的城堡离诺德林根(Nördlingen)不远，他和他的儿子们是一直值得依赖的谋臣，总是陪伴着年轻的斯陶芬。

但是，他一方面在士瓦本的神职人员找到了支持，同时在属臣们(Ministerialität)中找到了支持：这是一种政治阶层，曾经在他的祖先腓特烈一世时期大范围内形成，但后来在很多方面变动很大，两个阶层都关联着公爵的利益。通过授予最强大的士瓦本主教及其仆从对政府事务管理的决定性影响，人们赢得了对康拉丁事业的信心。另一方面，公爵的宫廷和可能的政治斗争为大臣们提供了一个施展的机会，他们更关心地位和战事而非经济利益。从这两个阶层中，我们能看到从他们的名字外无从了解的部分个人特征。

康拉丁的新监护人康斯坦茨的主教埃伯哈德(Eberhard von Konstanz)是一位好战的统治者。他的祖先托赫塞森·瓦尔德堡(Truchsessen von Waldburg)为霍亨斯陶芬王朝做出了重要贡献，但他本人却站在对立国王(Gegenkönige)和教皇一边。在这个混乱的年代，他反而进一步扩大了他的权力范围，起初他和圣加仑修道院长贝尔特霍尔德(Abbot Berthold von St. Gallen)激烈斗争，后又与他结成了紧密联盟。现在他们一起统治了波登湖(Bodensee)周围的土地，并在伊勒河上(Iller)发放安全通行证，康斯坦茨家族在那里毗邻奥格斯堡教区(Augsburger Diözese)。在这两个人身上，看不到新宗教的精神，这种精神当时充斥在人们心中并且在其他地方已经强势出现。在德国南部各地，雷根斯堡的贝尔特霍尔德(Berthold von Regensburg)以其震撼人心的露天苦修唤醒了罪人的良知，盖斯勒派(Geissler)第一次带着苦刑和祈祷穿越了德国南部，而多明我会修士阿尔贝图斯·马格努斯(Albertus Magnus)为了安静地整理所有中世纪的知识而放弃了雷根斯堡(Regensburger)主教，正如马纳(Marner)所抱怨的那样，“偷来的东西换成了剑，骗子变成了长矛”。他们的钻营只是为了“骑士和俗世的荣誉”，当然，他们彼此也臭味相投。

圣加仑修道院院长贝尔特霍尔德·法尔肯斯坦(Berthold von Falkenstein)的画像后来由一位修道院的编年史家绘制——可能带有个人的，作坊式的修饰和夸张，但本质上是真实的，并且有着入木三分的生动。不管一个人多么反感他的好勇斗狠和冷酷无情，都不得不由衷钦佩他罕见的政治才能，光彩的骑士风范和非凡的慷慨大方。如果节日在教堂里举办盛大的庆祝活动，周围聚集着众多骑士还有小提琴手演奏，那他就能在庆典期间立马引起斗殴并带领那坨骑士战斗，他就这么个人！可他知道如何在困难的情况下赢得教皇的青睐，通过战斗和购买来扩大修道院的领地，向上帝的子民征收丰厚的税款，还压制他们的愤怒直到最后自作孽不可活。只要他的体制不瓦解那他就能继续风光。最终他孤独地死去，被所有人仇恨，教堂的修士为他做弥撒，上帝的子民为他而欢呼。

康拉丁一直和这两个人保持着密切接触。他和他的主教监护人拜访了圣加仑修道院(Abbot Berthold von St. Gallen)，受到了周到的对待并呆了三天。可能就是在这个时候，居住在那里的温根的赫尔(Herr von Wengen)在一首歌中赞美了月亮，将为国家带来崇拜和荣耀的月亮。贝尔特霍尔德(Berthold von Falkenstein)的重要性增加了不少，因为他与士瓦本的家族有点姻亲关系，可以为康拉丁争取他们。从康斯坦茨教区(Konstanzer Sprengel)修道院的一些特权看，以及主教与院长在重要步骤的着重表述，经常性的共同密封一些文件，还有这些文件中的宗教规定，可以承认这两个人对当下与未来康拉丁的士瓦本公国产生的决定性影响。

除了他们之外，肯普滕的鲁伯特(Rupert von Kempten)院长位居他们之后。在康斯坦茨的那场出席人数众多的庭审日期间，康拉丁放弃了肯普滕(Kempten)修道院每年50马克的年金。这个领主权利不能说不重要，到底是因为什么呢，资金的需要？压过院长一头或是康斯坦茨主教的调解？都无法确定。奥格斯堡主教哈特曼(Hartmann von Augsburg)对康拉丁起初的影响很微小，直到后来康拉丁才不得不与他打交道，部分是敌对的，部分是友好的。

士瓦本的宫廷大臣，首先是几个最重要的行政首脑，帕彭海姆(Pappenheim)元帅，林堡(Limburg)王家酒侍(Reichsschenken)，接着是士瓦本宫廷官员瓦尔德堡(Waldburg)和瓦尔特豪森(Warthusen)的法院院长(Truchsessen)，施马尔内克(Schmalnecke)-温特施泰滕(Winterstetten)的酒侍(Schenken)，拉文斯堡(Ravensburg)和比恩堡(Bienburg)的司库(Kämmerer)，他们或早或晚的出现在康拉丁周围。他们的多种族，昂扬进取，骑士精神的导向，仆人的服务(Frauendienst)与诗歌，似乎决定了这个正在成长的男孩正在向好的方向发展。

自然而然地，康拉丁与自己的大臣们建立了更密切的个人关系，他们是他的忠实顾问，在某些情况下也是公爵政策的执行机构。可惜的是，我们对其他霍亨斯陶芬资产(Güter)的范围和管理信息知之甚少。除了极少的例外，士瓦本北部、巴登(Baden)和阿尔萨斯(Elsass)的旧斯陶芬资产以及在弗兰肯(Franken)的萨利(salische)遗产都已丢失。除了一些王室资产(Reichsgüter)的名号和班贝格(Bamberg)的封地外，留下来的基本都是莱希河(Lech)两岸，阿尔高(Allgäu)和波登湖(Bodensee)老旧房屋。上面提到的大臣几乎都来自这几个地区，我们知道的那些与康拉丁始终相伴的大臣和权贵也是如此：沃尔克马尔·凯姆纳滕(Volkmar von Kemnaten)、斯威克·明德尔伯格(Swiker von Mindelberg)和赫尔曼·冯·哈根伯格(Hermann von Hägenberg)——此外还有维特尔斯巴赫(Wittelsbach)大臣艾索兹里德的海因里希(Heinrich von Eisolzried)。年轻的公爵后来非常重视他们的建议，将重要的事务也让他们参与，甚至给了他们重大的管理职责。当然，最开始，路德维希公爵可能已经任命了巴伐利亚贵族布鲁克贝格的阿尔贝罗(Albero von Bruckberg)，他是康拉丁在莱希(Lech)的佩廷城堡(Burg Peiting)的钦差(Prokurator)，负责管理整个公国的司法和行政的部长。后来又委派沃尔克马尔·凯姆纳滕(Volkmar von Kemnaten)临时监管莱希(Lech)地区最重要的区域。

几乎找不到比沃尔克马尔这个人更合适康拉丁的首席导师(Haupterzieher)了，他在那个时代就已经获得同代人的认可和尊敬了。他知道如何将那个时代流行的侠义、艺术的情操与遇事的冷静、务实的精神完美地结合起来。喜欢争斗的人把他视作合格的战士，爱好和平的人把他视作法定的仲裁，他是流浪人慷慨的施舍人，在丰富的经验中他建立了深厚的人际关系，现在成为主人顾问的时机成熟了。他是乌尔滕合恩家族(Herrn von Ulten)的家臣(Dienstmann)，除了在蒂罗尔(Tirol)的资产外，他还通过继承获得了朗斯贝格侯爵(Markgrafen von Ronsberg)（在莱希(Lech)地区）的所有领地，他熟知蒂罗尔(Tirol)的情况，被迈因哈德伯爵(Grafen von Görz und Tirol)授予内臣(Vertrauensmann)，管理和顾问重要的事务。当腓特烈二世为自己家族获得这部分资产时，沃尔克马尔就与斯陶芬家形成了雇佣关系，而且经常出现在康拉德四世左右。他还获得了奥格斯堡主教(Bischofs von Augsburg)的帮助，主教让他作奥本舍费德(Oberschönefeld)修道院(Frauenkloster)的长官(Schirmvogtei)。他的继父以前是康斯坦茨主教埃伯哈德(Bischofs von Konstanz)的侍丛，他又从继父那里继承了波登湖旁的阿尔邦镇(Stadt Arbon)。阿尔邦在圣加仑修道院领地范围内，却过的不如意：想不出比这更有利于康拉丁的关系了。偶尔听闻沃尔克马尔(Volkmar)对福森(Füssen)修道院造成的破坏是事实，但我们用不着稀奇，那个时候最好的修道院也不可能没有破坏。康拉丁其实很感激沃尔克马尔的服务，他和他的儿子马夸德(Marquard)经常伴其左右，对奥本舍费德(Oberschönefeld)修道院的捐赠和一些重大使命的交托就可以看出来，而修道院的院长1正是沃尔克马尔(Volkmar)的女儿阿登海兹(Adelheid)。年轻的公爵在阿尔邦长时间逗留，他和他的随行人员应该在那里受到了沃尔克马尔的热情款待，这告诉我们他们之间亲密的个人关系。康拉丁后来在一份文件中为那段时光表示了友好，授予阿尔邦社区司法自治的权力。

1. 奥本舍费德修道院是一座女修道院(Frauenkloster) ↩
